DEEN DAYAL AWASIYA YOJNA - STATIC WEBSITE

Files:
- index.html
- styles.css
- script.js
- assets/ (project images, price list and supplied MP4 video)

HOW TO PREVIEW
1. Open index.html directly in a modern browser, OR
2. Run a local server from this folder:
   python -m http.server 8000
3. Visit http://localhost:8000

IMPORTANT
The site intentionally includes a clear 'Not a Government Website' notice and private-project disclaimer. Do not remove this if the website is publicly deployed under a government-like project name or visual style.
